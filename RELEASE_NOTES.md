# Home Launcher - AI 手机启动器

## 📱 应用介绍

Home Launcher 是一款创新的 Android 桌面启动器，专为打造"AI 手机"体验而设计。安装后，您的手机将变成一个以 AI 助手（豆包）为中心的智能设备。

---

## ✨ 核心特性

### 🤖 AI 优先体验
- **开机直达 AI**：设备启动后自动进入豆包语音助手
- **语音交互中心**：让 AI 成为手机的第一入口，而非传统应用列表
- **沉浸式体验**：专注对话，减少干扰

### 🔒 应用锁定模式
- **防止误操作**：锁定后无法随意退出 AI 界面
- **专注模式**：特别适合儿童、老人或专注工作场景
- **智能解锁**：通过音量键组合快速解锁，恢复正常使用

### ⚡ 极简设计
- **纯黑深色主题**：护眼省电
- **简洁应用列表**：只显示必要应用
- **流畅启动**：轻量级设计，占用资源极小

## 🚀 快速开始

### 安装步骤

1. **下载 APK**
   - 访问 GitHub Releases 页面下载最新版本
   - 或使用 ADB 安装：`adb install app-debug.apk`

2. **设为默认桌面**
   - 按手机的 Home 键
   - 选择 "Home Launcher"
   - 勾选"始终"→ 点击"确定"

3. **开启无障碍服务**（可选，用于完全锁定）
   - 设置 → 无障碍 → Home Launcher → 开启

4. **重启设备**
   - 享受 AI 手机体验！

### 解锁方法

| 方法 | 操作 |
|------|------|
| **音量键解锁** | 同时按住 音量+ 和 音量- |
| **点击解锁** | 回到桌面后连续点击屏幕 3 次 |
