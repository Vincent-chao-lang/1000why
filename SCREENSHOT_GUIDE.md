## 截图拍摄技巧

### 准备工作

1. **使用模拟器或真机**
   - Android Studio 模拟器：可截图无状态栏
   - 真机：更真实的效果

2. **清理环境**
   - 移除不必要的应用
   - 设置合适的白名单
   - 确保界面整洁

3. **截图工具**
   - **Android Studio**: Layout Inspector 或 Device File Explorer
   - **adb 命令**: `adb shell screencap -p /sdcard/screen.png`
   - **模拟器自带**: 截图功能
   - **第三方**: SCAPY 等工具

### 拍摄建议

| 技巧 | 说明 |
|------|------|
| 统一尺寸 | 所有截图保持相同尺寸 |
| 清晰度 | 使用最高分辨率 |
| 无干扰 | 隐藏状态栏和导航栏 |
| 突出重点 | 用红色圆圈标注关键区域 |
| 加文字 | 添加简短说明文字 |

## 快速截图命令

### 使用 ADB 截图

```bash
# 截图到电脑
adb shell screencap -p > screen1.png

# 截图到手机后拉取
adb shell screencap -p /sdcard/screen.png
adb pull /sdcard/screen.png
```

### 批量截图脚本

```bash
#!/bin/bash
# 快速截图 5 张

for i in {1..5}
do
    echo "拍摄截图 $i - 按回车继续..."
    read
    adb shell screencap -p > screenshot_$i.png
    echo "截图 $i 已保存"
done
```
